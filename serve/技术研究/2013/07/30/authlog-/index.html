auth.log 详细分析

  • 日志分析 4
  • log 5

文件基本信息

Under CentOS / RHEL / Fedora file name is /var/log/secure.Under ubuntu the file name is /var/log auth.log.

Cent OS release 5.4 文件权限是600.Ubuntu 12.04里权限是640.可见这个文件对权限要求较高。
