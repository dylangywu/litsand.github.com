emacs速查

  • emacs 1

替换

M-% (然后会出现Query replace的字样)

Query replace:<被替换字符串>

Query replace <被替换字符串> with:<替换字符串>

Query replacing <被替换字符串> with <替换字符串>:（? for help）

查询替换命令的选项如下：

y 或者 空格 键 替换搜索到的字符串

n 或者 Del 键 取消替换搜索到的字符串

^ 回到前一个搜索到的字符串

! 替换所有没有替换的与模式匹配的字符串

ESC 键 退出本次查询搜索

复制

M+w  复制
C+y  粘贴
